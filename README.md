# Post web app

### Run backend

Install node_modules

```terminal
    yarn install
```

Run backend server

```terminal
    yarn start:dev
```

GraphQL run on 

`http://localhost:8080/graphql`

### Run frontend

```terminal
    cd frontend
```
Install node_modules

```terminal
    yarn install
```

Run frontend server

```terminal
    yarn start
```
frontend run on 

`http://localhost:3000`
